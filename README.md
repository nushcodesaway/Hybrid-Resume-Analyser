# 🚀 Hybrid Resume Analyzer

## 📌 Overview

This project implements a **Hybrid Lexical-Semantic Matching System** for resume analysis.
It combines **TF-IDF (lexical matching)** and **Sentence Transformers (semantic matching)** to improve search accuracy.

---

## 🎯 Features

* 🔍 Hybrid Search (Lexical + Semantic)
* 🤖 Machine Learning (Clustering + Anomaly Detection)
* 📡 IoT Simulation (Behavioral Data)
* 📊 Resume Ranking Dashboard
* 🧠 Explainability (Top Resume Analysis)
* 📉 Skill Gap Analysis
* 📊 Visualization (Score Graph)

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* Sentence Transformers
* Pandas
* Matplotlib
* PDFPlumber

---

## ⚙️ How It Works

1. Upload resumes (PDF)
2. Enter job description
3. System:

   * Extracts and preprocesses text
   * Computes lexical + semantic similarity
   * Generates behavioral (IoT) data
   * Detects anomalies (Isolation Forest)
   * Clusters candidates (K-Means)
4. Displays ranked results with insights

---

## 📡 IoT Simulation

Behavioral data such as typing speed, time spent, and edit patterns are simulated to mimic real-world user interaction.

---

## 📊 Output

* Resume ranking
* Similarity scores
* Cluster labels
* Anomaly detection
* Skill gap analysis
